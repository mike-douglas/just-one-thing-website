# Just One Thing Website

This is the repo containing the website for the app Just One Thing.

Here are the high level requirements for the website:

- It's written in Vercel. The site should be built with this framework and follow the best practices defined in the vercel skills available in this project.
- The copy on the website follows the guidelines in the just-one-thing-product skill for color scheme, voice, and tone. The features in the app are defined in the skill as well.
- There are screenshots from the app for reference in `./screenshots/` containing the design/aesthetic, as well as overviews of the features. These are ONLY for reference; you will add placeholders for the actual screenshots that may appear in the website.
- There are ASO screenshots in `./aso-screenshots` that contain the screenshots used in the App Store listing for reference.

## Design specifics for the website

- The site will be designed in a narrow single column view that is mobile responsive. On desktop, the single column will be relatively small and center-aligned. The logotype (logotype.svg will be provided) will show prominently at the top. On mobile the design will span the width of the screen.
